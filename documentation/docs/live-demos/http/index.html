Directed Percolation demonstration

This is a demonstration of the directed percolation library, built in Rust, using a 'WASM' wrapper and so running in a browser

This utilizes the 'directed_percolation' Rust crate, which is also used by the Python wrapper and the python notebooks; however, 'WASM' provides only a single thread, and so the simulations that are performed in the browser are all single threaded, and much lower performance than native simulations - so the scale of simulations within the browser is somewhat limited.
